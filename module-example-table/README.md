mod_example_table
=================

This is an example module which demonstrates the use of the [Joomla Table class](https://manual.joomla.org/docs/general-concepts/table).

See the Sample Module sections in [Basic Table Functionality](https://manual.joomla.org/docs/general-concepts/table/basic-table) and
[Advanced Table Functionality](https://manual.joomla.org/docs/general-concepts/table/advanced-table) 
in the Joomla Manual for a detailed description of how to use it.
